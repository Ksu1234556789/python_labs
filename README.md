## Course
### В работе реализован пользовательский класс Course, который моделирует учебный курс в образовательной системе. Класс содержит атрибуты объекта: название курса, преподавателя, количество часов и количество студентов. Также реализовано логическое состояние курса (активен или закрыт).

## Атрибуты класса
- MAX_STUDENTS — максимальное количество студентов на курсе
- MIN_HOURS — минимальное количество часов курса
- existing_courses — хранит уже созданные курсы (уникальность)

## Закрытые поля:
- _title — название курса
- _teacher — преподаватель курса
- _hours — количество учебных часов
- _students_count — количество студентов
- _active — состояние курса (активен / закрыт)

## Свойства (@property):
- Чтение: title — название курса
- Чтение: teacher — преподаватель
- Чтение: hours — количество часов
- Чтение и запись: students_count — количество студентов
- Чтение: active — состояние курса

## Магические методы:
- __str__ — удобный вывод информации о курсе
- __repr__ — техническое представление объекта
- __eq__ — сравнение курсов по названию и преподавателю

## Бизнес-методы:
- add_student() — добавить студента на курс
- remove_student() — удалить студента
- close() — закрыть курс
- open() — открыть курс
